# Errors

| Error | Code | Description |
|---|---|---|
| ErrDenomExists | 2 | Denom already has bank metadata |
| ErrUnauthorized | 3 | Sender is not the denom admin |
| ErrInvalidDenom | 4 | Invalid denom format |
| ErrInvalidCreator | 5 | Invalid creator address |
| ErrInvalidAuthorityMetadata | 6 | Invalid authority metadata |
| ErrInvalidGenesis | 7 | Invalid genesis state |
| ErrSubdenomTooLong | 8 | Subdenom exceeds 44 bytes |
| ErrCreatorTooLong | 9 | Creator address exceeds 75 bytes |
| ErrDenomDoesNotExist | 10 | Denom not found |
| ErrCapabilityNotEnabled | 11 | Capability disabled on chain |
