# Errors

| Error | Code | Description |
|---|---|---|
| ErrWrongName | 2 | name length not 1–32 characters |
| ErrRouteNotExist | 3 | route does not exist |
| ErrRouteExist | 4 | route already exists |
| ErrWrongValueDenom | 5 | denom is not millivolt or milliampere |
| ErrMaxRoutes | 6 | max routes exceeded |
| ErrSelfRoute | 7 | routing to self is not allowed |
